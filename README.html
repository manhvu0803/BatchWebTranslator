# How to Use Type any text in any language into the text box and click Translate to start translating
Once the translating is done, clicked Copy text to copy all the translated sentences, delimited by tab
Then paste into any text editor of your choice # Options ## Copy for Excel Make the copied text readable by Excel, WPS and other spreadsheet editors
Uncheck this if you are copying the translation directly into a raw text file ## HTML tags You can use any html tags (``, ``, ``, etc.) in the text ## Quick coloring Use 2 brackets: ((text)) to insert color tag around the text. The tag color is the one you choose in Replace color ## Untranslate phrase Use 2 minus signs: `--text--` to mark parts as ignored when translating


# GPT specific ## Temperature Change chatGPT setting (0 means less risk-taking and more stable translation, 1 means more random translation) ## Tone Try to influence chatGPT choice of words. Might change the sentence meaning ## Error Make chatGPT check for errors and correct them before translating. Usually take twice the time to translate
