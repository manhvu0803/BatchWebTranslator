# All ## HTML tags This tool support all html tags (``, ``, ``, etc.) ## Quick color Use 2 brackets: ((text)) to insert color tag around the text. The tag color is the one you choose in Replace color ## Untranslate phrase Use 2 minus: `--text--` to mark parts as ignored when translating # GPT specific ## Temperature Change chatGPT setting (0 means less risk-taking and more stable translation and vice versa) ## Tone Try to influence chatGPT choice of words. Might change the sentence meaning ## Error Make chatGPT check for errors and correct them before translating. Usually take twice the time to translate
